# Chapter 11: 시스템 (Systems)

이번 장에서는 높은 추상화 수준, 즉 시스템 수준에서도 깨끗함을 유지하는 방법을 살펴본다. 시스템 수준에서 깨끗함을 유지하는 것은 시스템의 아키텍처와 디자인을 신중하게 고려해야 한다는 것을 의미한다. 시스템이 복잡해질수록, 잘못된 설계 결정이 전체 시스템에 영향을 미칠 수 있기 때문에, 초기 단계에서 올바른 아키텍처를 선택하는 것이 중요하다.
