# Chapter 2. Meaningful Names (의미 있는 이름 짓기)

이 장에서는 이름을 잘 짓는 간단한 규칙을 소개한다. 좋은 이름은 읽기 쉽고, 이해하기 쉽고, 검색하기 쉽다.
