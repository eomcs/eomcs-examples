## 기본 문자열 메서드

유용한 문자열 메서드가 많이 있습니다. 예를 들어, `lower()` 메서드를 사용하여 문자열에서 대문자를 제거할 수 있습니다. `upper()` 메서드는 문자열을 대문자로 변환하는 데 사용됩니다. 문자열 메서드를 호출하려면, 문자열(또는 문자열을 포함하는 변수) 뒤에 점을 입력한 다음 메서드 이름을 입력하면 됩니다. 예: `"John".upper()` PyCharm에서는 점 뒤에 &shortcut:CodeCompletion; 단축키를 사용하여 사용 가능한 모든 문자열 메서드를 탐색할 수 있습니다. 모든 문자열 메서드에 대한 자세한 정보는 <a href="https://docs.python.org/3/library/stdtypes.html#string-methods">여기</a>에서 확인할 수 있습니다.

더 체계적이고 자세한 정보는 [이 Hyperskill 지식 기반 페이지](https://hyperskill.org/learn/step/6842?utm_source=jba&utm_medium=jba_courses_links)를 참조하십시오.
