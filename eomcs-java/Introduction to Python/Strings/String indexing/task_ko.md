## 문자열 인덱싱

문자열에서 위치를 알고 있다면 해당 위치의 문자를 접근할 수 있습니다. 예를 들어,  
`str[index]`는 문자열 `str`에서 `index` 위치에 있는 문자를 반환합니다.  
문자열 인덱싱은 항상 `0`부터 시작한다는 점에 유의하세요.  
`index`는 문자열에 `x`가 없을 경우 `ValueError`를 발생시킵니다.  

인덱스는 문자열의 오른쪽 끝(즉, 문자열 끝부분)에서부터 세야 할 경우 음수일 수도 있습니다.  
참고로 `-0`은 `0`과 동일하기 때문에, 음수 인덱스는 `-1`부터 시작합니다.
