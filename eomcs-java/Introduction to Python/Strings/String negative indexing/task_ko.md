## 문자열에서 음수 인덱싱

앞서 말씀드린 것처럼, 문자열의 끝에서부터 '거꾸로' 문자를 세기 위해 음수 인덱스를 사용할 수 있습니다.

더 구조적이고 자세한 정보는 [Hyperskill 지식 기반 페이지](https://hyperskill.org/learn/step/6189?utm_source=jba&utm_medium=jba_courses_links)를 참조하세요.

### 과제
음수 인덱스를 사용하여 `long_string`에서 `'!'` 기호를 가져오세요.
