## 슬라이싱

슬라이싱은 문자열에서 여러 문자(서브스트링)를 추출하는 데 사용됩니다.  
구문은 인덱싱과 유사하지만, 하나의 인덱스 대신 콜론으로 구분된 두 개의 인덱스(숫자)를 사용합니다. 예: `str[ind1:ind2]`.  
이 두 인덱스는 서브스트링의 시작과 끝을 나타냅니다. 주의할 점은 인덱스 `ind1`에 해당하는 문자는 포함되지만,  
인덱스 `ind2`에 해당하는 문자는 포함되지 않는다는 것입니다.

다음은 Python에서 슬라이싱이 작동하는 방식을 시각적으로 나타낸 것입니다:

```text
+---+---+---+---+---+---+
| P | y | t | h | o | n |
+---+---+---+---+---+---+
0   1   2   3   4   5   6
-6  -5  -4  -3  -2  -1
```

##### 예제
<pre><code>
str[start:end] # start부터 end-1까지의 요소
str[start:]    # start부터 배열의 끝까지의 요소
str[:end]      # 시작부터 end-1까지의 요소
str[:]         # 배열 전체의 복사본
</code></pre>
