## 문자열 포매팅

문자열 뒤의 `%` 연산자는 문자열과 변수를 결합하는 데 사용됩니다.  
`%` 연산자는 문자열 내 `%s`를 뒤따르는 문자열 변수로 대체합니다.  
`%d` 특별 기호는 숫자나 소수 값을 위한 자리 표시자로 사용됩니다.  

> <i><b>참고</b>: 여기에서 설명된 포매팅 연산은 여러 가지 특이점을 보여주며,  
> 이는 일반적인 오류를 일으킬 수 있습니다.  
> 보다 새로운 형식 문자열 리터럴 (참고: [더 알아보기](course://Strings/F-strings)),  
> <code><a href="https://docs.python.org/3/library/stdtypes.html#str.format">str.format()</a></code> 인터페이스 또는  
> <a href="https://docs.python.org/3/library/string.html#template-strings">템플릿 문자열</a>을 사용하면 이러한 오류를 방지할 수 있습니다.  
> 이러한 대안 각각은 단순성, 유연성 및/또는 확장성 측면에서 고유한 장단점을 제공합니다.</i>
