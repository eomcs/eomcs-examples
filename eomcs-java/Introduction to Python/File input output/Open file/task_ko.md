## 파일 열기  
Python은 컴퓨터의 파일에서 정보를 읽고 쓰기 위한 여러 내장 함수를 제공합니다.

`open()` 함수는 파일 객체를 반환하며, 가장 일반적으로 두 개의 인자와 함께 사용됩니다: `open(filename, mode)`:
```python
f = open('somefile.txt', 'w')
```
첫 번째 인자는 파일 이름을 포함하는 문자열입니다. 두 번째 인자는 파일이 사용될 방식을 설명하는 몇 가지 문자를 포함하는 또 다른 문자열입니다.  
파일을 읽기 전용으로 열 때는 `'r'`, 쓰기 전용으로 열 때는 `'w'` (동일한 이름의 기존 파일은 삭제됨),  
그리고 `'a'`는 파일을 추가 모드로 열어 데이터를 파일 끝에 추가하도록 합니다. `'r+'`는 파일을 읽기와 쓰기 둘 다 가능하도록 엽니다. `mode` 인자는 선택 사항이며, 생략될 경우 기본값은 `'r'`로 설정됩니다.

파일 객체를 다룰 때는 `with` 키워드를 사용하는 것이 좋은 습관입니다. 이를 통해 코드 블록 실행이 완료되면 파일이 적절하게 닫히는 이점을 제공합니다.

```python
with open('somefile.txt') as f:
    read_data = f.read()

# 파일이 자동으로 닫혔는지 확인할 수 있습니다.
f.closed
```
```text
True
```
**중요**: `with` 키워드를 사용하지 않는 경우, 반드시 `f.close()`를 호출하여 파일을 닫고 사용한 시스템 리소스를 해제해야 합니다.  
`with` 문이나 `f.close()` 호출로 파일이 닫힌 후에는 해당 파일 객체를 사용할 수 없습니다.
