### 다음 단계는?
Python 프로그래밍의 기초를 탐구하셨으니, 이제 더 깊이 배우고 새로 얻은 기술을 연마할 차례입니다! 다음 단계로 고려할 옵션들은 다음과 같습니다:
- Python에서 가장 널리 사용되는 라이브러리를 배우려면, 우리의 [NumPy 과정](https://plugins.jetbrains.com/plugin/18302-python-libraries--numpy)을 참고하세요.
- Python으로 [AMazing](https://plugins.jetbrains.com/plugin/17519-amazing) 미로를 만드는 방법을 알아보세요.
- [기계 학습](https://plugins.jetbrains.com/plugin/18392-machine-learning-101)의 기초를 발견해보세요.
- JetBrains Academy에서 Hyperskill의 [Python Core](https://hyperskill.org/tracks/2) 또는 [Python for Beginners](https://hyperskill.org/tracks/6) 트랙의 프로젝트를 확인하세요.

또한 [JetBrains Academy 카탈로그](https://academy.jetbrains.com/)에서 수많은 다른 과정들을 찾을 수 있습니다!

새로운 과정과 업데이트를 기대해주세요!
