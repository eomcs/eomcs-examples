## 모듈 2 가져오기

함수뿐만 아니라 클래스나 다른 모듈도 가져올 수 있습니다. 필수는 아니지만, 모든 `import` 구문을 모듈의 맨 처음에 배치하는 것이 관례입니다.

Python의 모듈에 대해 더 알고 싶다면 [이 섹션](https://docs.python.org/3/tutorial/modules.html)을 참고하세요.

### 과제
코드 편집기에서 `calculator` 모듈을 가져오고 `Calculator` 클래스의 인스턴스(`calc`)를 생성하세요.  
`Calculator`에 정의된 `add` 메서드를 루프에서 사용하여 0부터 99까지 숫자를 더하세요.

<div class='hint'><code>import</code> 키워드와 <code>calculator</code> 참조를 사용하세요.</div>
<div class='hint'><code>module.function()</code>과 같은 문법으로 모듈에서 함수를 접근하세요.</div>
<div class="hint">함수에 인수를 제공하는 것을 잊지 마세요.</div>
