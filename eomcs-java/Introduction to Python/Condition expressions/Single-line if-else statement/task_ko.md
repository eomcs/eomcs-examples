이전 과제에서 보았듯이, 간단한 `if-else` 문은 사실 단일 라인으로 작성될 수 있습니다.

### 과제
코드 편집기에서의 문장을 수정하여
단일 라인에 맞추되 실행 결과는 정확히 동일하게 유지하세요:
- `my_random_number`가 50보다 클 경우, 프로그램은 `my_random_number`의 값을 출력해야 합니다.
- 그렇지 않으면, `"Too small!"`이라는 메시지를 출력해야 합니다.
