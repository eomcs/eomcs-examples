## 불린 연산자

불린 연산자는 명제를 비교하고 불린 값으로 결과를 반환합니다. 불린 연산자 `and`는 `and`의 양쪽 표현식이 모두 `True`일 때 `True`를 반환합니다. 불린 연산자 `or`는 `or`의 어느 한쪽 표현식이라도 `True`일 경우 `True`를 반환합니다. 불린 연산자 `not`은 그 앞에 오는 불린 표현식을 반전시킵니다.  

보다 체계적이고 자세한 정보는 [이 Hyperskill 지식 기반 페이지](https://hyperskill.org/learn/step/6025?utm_source=jba&utm_medium=jba_courses_links)를 참고하세요.

### 과제
`name`이 `"John"`과 같고 그가 `16`세 이상이면 `True`로 평가되는 표현식을 작성하세요.

<div class='hint'><code>and</code> 키워드와 <code>>=</code> 연산자를 사용하세요.</div>
