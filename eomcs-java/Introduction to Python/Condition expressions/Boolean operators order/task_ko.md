## 불리언 연산자의 우선순위

불리언 연산자는 왼쪽에서 오른쪽 순서로 평가되지 않습니다. 불리언 연산자에는 연산 순서가 있습니다: `not`이 먼저 평가되고, 그다음 `and`, 마지막으로 `or`이 평가됩니다.

보다 구조적이고 자세한 정보는 [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/6025?utm_source=jba&utm_medium=jba_courses_links)를 참조하십시오.

### 작업
`name`이 `"John"` 또는 `"Jane"`이고, `16`세 이상 `25`세 미만일 경우 `True`로 평가되는 표현식을 작성하세요.

<div class='hint'><code>and</code>와 <code>or</code> 키워드를 결합하세요.</div>
