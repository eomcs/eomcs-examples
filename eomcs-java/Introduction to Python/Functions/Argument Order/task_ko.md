## 인수 순서

코드 편집기에는 주어진 행과 열 수로 테이블을 출력하는 함수가 있습니다. 또한 이 함수는 테이블 출력에 사용할 다양한 기호를 사용할 수 있습니다. 아래 함수 호출에는 몇 가지 오류가 포함되어 있습니다. 코드를 수정하여 아래와 같은 테이블을 출력하십시오:

```text
|____|____|____|____|____|

|____|____|____|____|____|

|____|____|____|____|____|

|____|____|____|____|____|

|____|____|____|____|____|

```

답안 자리에서 코드만 수정하면 됩니다!
