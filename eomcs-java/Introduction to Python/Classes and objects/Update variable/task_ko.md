## 변수 업데이트

이 연습에서는 새로운 `Car` 객체 `car2`를 생성한 다음, `car2`의 `color`를 `"red"`로 설정하세요. 그리고 `description` 메서드를 호출하여 `car1`과 `car2`의 설명을 출력하세요. 

더 구조적이고 자세한 정보를 원한다면 [이 Hyperskill 지식 기반 페이지](https://hyperskill.org/learn/step/6677#changing-attributes?utm_source=jba&utm_medium=jba_courses_links)를 참조하십시오.

<div class='hint'>이전 과제에서 클래스 "호출"을 통해 객체를 인스턴스화했던 방법을 기억하세요.</div>
<div class='hint'>값 할당 (<code>=</code>)과 속성 참조 문법 <code>obj.attribute</code>을 사용하여 색상을 변경하세요.</div>
<div class='hint'>출력문에서 이전 과제에서 했던 것처럼 <code>description()</code> 메서드를 호출하세요.</div>
