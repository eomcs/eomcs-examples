## 다른 메서드에서 메서드 호출하기

앞서 논의했듯이, 메서드는 `self` 인수의 메서드 속성을 사용하여 다른 메서드를 호출할 수 있습니다.  
이 지식을 활용하여 `Calculator` 클래스의 `exponentiate` 메서드를 완성하세요.  
이 메서드는 결과를 계산하기 위해 `multiply` 메서드를 호출해야 합니다.
