## 클래스와 인스턴스 변수 2  

### 작업  

`City` 클래스에, 모든 추가된 도시 이름을 저장하기 위한 변경 가능한 클래스 변수인 리스트 `all_cities`를 추가하세요.  

`add_city` 메서드를 구현하여 `all_cities` 리스트에 도시 이름을 추가하세요. `pass` 문을 삭제하는 것을 잊지 마세요!  

`__init__` 메서드에서는 객체가 초기화될 때 도시 이름을 리스트에 추가하기 위해 `add_city` 메서드를 호출하세요.
