## 정의

클래스는 데이터와 기능을 함께 묶는 수단을 제공합니다. 새로운 클래스를 생성하면 새로운 유형의 객체를 만들 수 있으며, 해당 유형의 새로운 인스턴스를 생성할 수 있습니다. 클래스는 본질적으로 객체를 생성하기 위한 템플릿입니다.  
각 클래스 인스턴스(객체)는 상태를 유지하기 위해 속성을 부여받을 수 있습니다. 객체의 함수는 메서드라고 하며 이들은 객체의 상태를 수정할 수 있습니다. 메서드는 객체의 클래스에서 정의됩니다.

클래스 정의의 일반적인 형식은 다음과 같습니다:

```
class ClassName:
    <statement-1>
    .
    .
    .
    <statement-N>
```
클래스 정의는 함수 정의(`def` 문)와 마찬가지로 실행되기 전까지는 아무런 효과도 없습니다.

클래스 정의 내부에는 일반적으로 함수 정의가 포함되지만, 때로는 다른 문이 유용할 수 있습니다. 클래스 내부의 함수 정의는 특유의 인자 목록 형식을 가지며, 이에 대해서는 나중에 설명됩니다.

클래스 객체는 두 가지 종류의 작업을 지원합니다: 속성 참조와 인스턴스화입니다. 속성 참조에 대해서는 이후 섹션에서 논의할 것입니다. 클래스 인스턴스화는 함수 표기법을 사용합니다. 클래스 객체가 매개변수가 없는 함수처럼 동작하여 새로운 클래스 인스턴스를 반환한다고 상상해 보십시오. 예를 들어:

```python
class SomeClass:
    """간단한 예제 클래스"""
    i = 12345

x = SomeClass()
```

`x = SomeClass()`는 클래스의 새로운 인스턴스를 생성하고 이 객체를 지역 변수 `x`에 할당합니다.

클래스 정의 문법에 대한 더 많은 정보를 원한다면 Python 공식 문서의 <a href="https://docs.python.org/3/tutorial/classes.html#class-definition-syntax">이 섹션</a>을 읽어보세요.
