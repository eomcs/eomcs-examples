## 변수 정의

변수는 값을 저장하여 나중에 참조할 수 있도록 합니다. 변수는 일종의 라벨과 같으며, Python에서는 ' `=` ' 기호를 사용하여 값을 변수에 할당합니다. 이는 할당 연산자라고 불립니다. 할당은 체인 형태로도 가능하며, 예를 들어 `a = b = 2`와 같이 쓸 수 있습니다.

### 작업
1. 문자열 `World`를 `name` 변수에 할당하세요.
2. 10번째 줄에서 `name` 변수의 값을 여러분의 실제 이름으로 바꾸세요.
3. 15번째 줄에서 체인 할당을 사용하여 값 2를 `a`와 `b` 모두에 저장하세요.
