## 할당

증강 할당(Augmented Assignment)은 이항 연산과 할당 문을 결합한 단일 문으로, `+=`, `-=` 등과 같은 연산자를 포함합니다.  

`x += 1`과 같은 증강 할당 표현은 `x = x + 1`로 다시 작성하여 유사한 효과를 얻을 수 있습니다.  
자세한 내용은 <a href="https://docs.python.org/3/reference/simple_stmts.html#augmented-assignment-statements">여기</a>에서 읽어볼 수 있습니다.

### 작업
증강 할당을 사용하여 `number`에 `5`를 더하고 변수를 업데이트하세요.
