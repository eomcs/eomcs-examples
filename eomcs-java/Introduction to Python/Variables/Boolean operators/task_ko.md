## 불리언 연산자

불리언(Boolean)은 `True` 또는 `False`만 가질 수 있는 값의 종류입니다. `==` (동등성) 연산자는 두 변수를 비교하여 동일한지 확인합니다. 불리언 연산자에 대해 더 자세히는 [과제](course://Condition expressions/Boolean operators)에서 배우게 될 것입니다. 

더 체계적이고 자세한 정보는 [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/6025?utm_source=jba&utm_medium=jba_courses_links)를 참조하세요.

### 과제
 - 변수 `two`가 `three`와 같은지 확인하세요.
 - 변수 `is_equal`이 오해를 불러일으키는 이름인지 확인하세요.
 - 변수 `is_false`가 정말로 거짓을 나타내는지 확인하세요.

<div class='hint'><code>==</code> 연산자를 사용하세요.</div>

<div class='hint'><code>True</code> 값을 사용하세요.</div>

<div class='hint'><code>False</code> 값을 사용하세요.</div>
