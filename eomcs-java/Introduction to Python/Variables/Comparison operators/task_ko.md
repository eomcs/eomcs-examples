## 비교 연산자

Python에는 여러 유형의 비교 연산자가 있습니다. 다음을 포함하여:  
- `<` 작다  

- `>` 크다  

- `==` 같다  

- `>=` 크거나 같다  

- `<=` 작거나 같다  

- `!=` 같지 않다

이 연산자들은 두 객체의 값을 비교합니다. 객체는 동일한 타입일 필요가 없습니다.

Python에서 모든 비교 연산은 동일한 우선순위를 가지며, 이는 어떠한 산술, 비트 이동, 또는 비트 연산보다 낮습니다. 비교는 부울 값, 즉 `True` 또는 `False`를 반환합니다. 비교는 임의로 연결될 수 있으며, `a < b < c`와 같은 표현은 일반적인 수학적 의미를 가집니다. 비교에 대해 더 알아보려면 <a href="https://docs.python.org/3/reference/expressions.html#comparisons">여기</a>를 참조하세요.

더 구조적이고 구체적인 정보를 얻으려면 [이 Hyperskill 지식 기반 페이지](https://hyperskill.org/learn/step/5920?utm_source=jba&utm_medium=jba_courses_links)를 참조할 수도 있습니다.

### 과제  
 - 변수 `three`의 값이 변수 `two`의 값보다 엄격히 큰지 확인하세요.  
 - 변수 `one`의 값이 변수 `three`의 값보다 엄격히 작은지 확인하세요.  
 - 변수 `three`의 값이 변수 `two`와 변수 `one`의 값보다 모두 엄격히 큰지 확인하세요.  
 - 변수 `one`이 변수 `two`와 같지 않은지를 확인하세요.  
 - 변수 `three`가 자기 자신과 같은지를 확인하세요.
