## 정의되지 않은 변수

변수 이름은 라틴 문자, 숫자 및/또는 밑줄 문자만 포함할 수 있으며, 숫자로 시작할 수 없습니다. 또한, 변수 이름은 <a href="https://docs.python.org/3/reference/lexical_analysis.html#keywords">예약어</a>가 될 수 없습니다.

### 작업
아직 정의되지 않은 변수를 사용하면 어떤 일이 발생하는지 확인하세요. 편집기에서 코드를 수정하고 실행하여 **정의되지 않은** 이름을 출력해 보세요 (즉, `print` 문 안의 괄호에 정의되지 않은 변수 이름을 입력).  
이 작업은 예외 - `NameError`가 발생해야 합니다.

"check" 명령어는 테스트 시스템만 실행하며 출력 결과를 볼 수 없으므로 파일을 실행해야 합니다.

<div class="hint">예를 들어, <code>variable</code> 대신 <code>var1</code> 변수를 출력하려고 해보세요.</div>
