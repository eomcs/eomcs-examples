## 타입 변환

여러 데이터를 한 타입에서 다른 타입으로 변환할 수 있는 내장 함수들이 있습니다.  
이 함수들은 변환된 값을 나타내는 새로운 객체를 반환합니다. `int(x)`는 `x`를 정수로 변환합니다.  
`float(x)`는 `x`를 부동소수점 숫자로 변환합니다. `str(x)`는 객체 `x`를 문자열 표현으로 변환합니다.

보다 체계적이고 자세한 정보를 원한다면, [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/6224?utm_source=jba&utm_medium=jba_courses_links)를 참조하세요.

### 과제
`float_number`를 정수로 변환하세요.
