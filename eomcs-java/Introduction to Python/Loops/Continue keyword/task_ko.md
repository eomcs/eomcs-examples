## continue 키워드

`continue` 키워드는 현재 반복의 나머지 코드를 건너뛰고 바로 다음 반복으로 진행할 때 사용됩니다. 루프는 `for` 또는 `while` 문으로 돌아가 조건을 다시 평가합니다. 만약 다음 항목이 없다면, 존재할 경우 `else` 절이 실행됩니다.

더 구조적이고 자세한 정보는 [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/6302#continue?utm_source=jba&utm_medium=jba_courses_links)를 참조하세요.

### 과제
홀수 `1, 3, 5, 7, 9`만 출력하세요.
