## for 루프

`for` 문은 문자열, 튜플, 리스트와 같은 시퀀스 또는 다른 iterable 객체의 요소를 순회(iterate)하는 데 사용됩니다. 시퀀스는 한 번 평가됩니다.  
각 순회(iteration)마다, `for` 루프에서 정의된 변수는 리스트의 다음 값이 할당됩니다.  
`for` 문이 있는 라인의 바로 아래에 있는 코드는 각 항목에 대해 한 번 실행됩니다.  
모든 항목이 소진되면 루프는 종료됩니다.

`for` 문에 대한 자세한 내용은 <a href="https://docs.python.org/3/reference/compound_stmts.html#the-for-statement">Python 문서의 이 페이지</a>에서 읽어볼 수 있습니다.
