## while 문

`while` 문은 `if` 문과 약간 비슷합니다: 어떤 조건이 `True`일 경우 일부 코드를 실행합니다. 주요 차이점은 조건이 `True`인 동안 들여쓰기된 코드를 계속 실행한다는 점입니다. 만약 표현식이 `False`라면, 루프가 종료됩니다.

보다 체계적이고 자세한 정보는 [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/5940?utm_source=jba&utm_medium=jba_courses_links)를 참고하세요.

### 과제  
`1`에서 `9`까지의 모든 제곱수 `(1, 4, ... , 81)`를 출력하세요. `while` 문에서 `number` 변수를 사용하세요.  

<div class='hint'><code>number</code>가 조건에서 10보다 엄격히 작은 값임을 확인하세요.</div>
