## break 키워드

무한 루프는 절대 종료되지 않는 루프를 말합니다. 루프 조건이 항상 `True`라면 이런 루프는 무한 루프가 됩니다. 무한 루프를 생성하는 가장 간단한 방법은 `while True`를 사용하는 것입니다. `break` 키워드는 루프를 벗어나기 위해 사용됩니다.  

더 구조적이고 자세한 정보를 보려면 [이 Hyperskill 지식 베이스 페이지](https://hyperskill.org/learn/step/6302#break?utm_source=jba&utm_medium=jba_courses_links)를 참조하세요.

### 작업
`animal`이 `elephant`인지 평가한 후 세 번째 반복 시 루프를 종료하는 조건을 작성하세요. `break` 문을 사용하세요. 마지막에 `zoo` 리스트에는 `lion`과 `tiger`만 포함되어 있어야 합니다(루프에서 나머지를 모두 `pop`하기 때문입니다).

<div class="hint"><code>animal</code>이 <code>"elephant"</code>과 같은지 확인하려면 <code>==</code>를 사용하세요.</div>
