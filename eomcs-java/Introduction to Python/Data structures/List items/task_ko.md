## 리스트 항목

리스트의 개별 요소에 할당할 수 있는 것처럼, 리스트의 슬라이스에도 할당할 수 있습니다.  
이를 통해 리스트의 크기를 변경하거나, 다음과 같이 리스트를 완전히 비울 수도 있습니다:

```python
animals[:] = []
```

### 과제
마지막 두 항목을 교체하여 `animals`를 모두 코끼리로 만드세요.  
먼저 IDE에서 프로그램을 아무런 변경 없이 실행해 보시는 것을 권장합니다. 이를 통해 모든 작업이 수행된 후 `animals` 리스트가 어떻게 보이는지 확인할 수 있습니다.

<div class='hint'>예제처럼 슬라이스에 할당을 사용하세요.</div>

<div class='hint'>여러 요소로 구성된 리스트를 할당할 수 있습니다.</div>
