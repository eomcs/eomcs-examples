## 중첩 리스트

리스트는 다른 리스트(하위 리스트)와 같은 모든 종류의 객체를 포함할 수 있습니다. 이러한 데이터 구조는 중첩 리스트(nested list)라고 불립니다.

중첩 리스트를 사용하여 데이터를 계층적 구조로 배열할 수 있습니다.

중첩 리스트는 쉼표로 구분된 하위 리스트들의 시퀀스를 작성하여 생성할 수 있습니다:

```python
nested_list = [[1, 2, 3], [4, 5], 6]
```

중첩 리스트에서는 이전과 동일하게 인덱스를 사용하여 항목에 접근할 수 있습니다:

```python
print(nested_list[1])
print(nested_list[2])
```
출력:
```text
[4, 5]
6
```
중첩 리스트 내 하위 리스트의 항목에 접근하려면 여러 인덱스를 사용할 수 있습니다.  
`nested_list`에서 숫자 `1`에 접근하려면 인덱스 `0`을 두 번 사용합니다. 먼저 요소 `[1,2,3]`에 접근한 뒤, 해당 하위 리스트의 첫 번째 요소에 접근합니다:
```python
print(nested_list[0][0])
```
출력:
```text
1
```
