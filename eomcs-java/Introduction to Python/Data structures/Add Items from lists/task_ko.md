## 리스트에서 사전 항목 추가하기

두 개의 리스트와 빈 사전을 미리 정의해 두었습니다. 여러분의 과제는 [`for` 반복문](course://Loops/For loop)을 사용하여 리스트를 반복(iterate)하고, 해당 리스트 항목들을 키와 값 쌍(key:value pair)으로 변환해 사전에 추가하는 것입니다. (예: key1: 100, key2: 200, key3: 300)  
`for` 문장은 이미 작성되어 있으니, 각 키와 값 쌍에 대한 할당문만 작성하면 됩니다.

<div class="hint">

`list_name[index]`와 같은 리스트 인덱싱 및 `dict[key] = value`와 같은 사전 인덱싱을 사용하여 값을 할당하세요.
</div>
