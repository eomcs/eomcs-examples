## 우리의 첫 번째 프로그램

전통적으로 어떤 프로그래밍 언어에서든 첫 번째로 작성하는 프로그램은 `"Hello World!"`입니다.  

세상에 본인을 소개해 보세요.  

이 스크립트를 실행하려면 **Editor** 보기의 아무 곳이나 마우스 오른쪽 클릭하여 컨텍스트 메뉴를 표시하고 **Run 'hello_world'**를 선택하세요.  
또는 &shortcut:RunClass; 단축키를 사용할 수도 있습니다.

자세한 내용은 [도움말](https://www.jetbrains.com/help/pycharm/running-and-rerunning-applications.html)을 확인하세요.
