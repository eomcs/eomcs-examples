## 주석

Python에서 주석은 해시 문자 (`#`)와 단일 공백으로 시작하며, 물리적 줄의 끝까지 이어집니다. PyCharm에서는 &shortcut:CommentByLineComment;를 사용해 한 줄 또는 코드 블록 전체를 주석 처리하거나 주석을 해제할 수 있습니다.  

코드가 변경될 때 주석도 항상 최신 상태로 유지하는 것을 우선시하세요! 코드를 모순되게 하는 주석은 주석이 없는 것보다 더 나쁩니다. 또한, 당연한 내용을 설명하는 주석은 불필요하며 오히려 방해가 될 수 있습니다. 이런 식으로 작성하지 마세요:

```python
x = x + 1                 # x 증가
```

주석은 완전한 문장으로 작성되어야 합니다. 첫 단어는 소문자로 시작하는 식별자가 아닌 이상 대문자로 시작해야 합니다. 주석은 명확하고 다른 사람들이 쉽게 이해할 수 있도록 작성하세요.  

#### 블록 주석

블록 주석은 일반적으로 뒤따르는 일부 (또는 전체) 코드에 적용되며, 해당 코드와 동일한 레벨로 들여쓰기 됩니다.

#### 인라인 주석  
인라인 주석은 가급적 적게 사용하세요. 인라인 주석은 명령문과 같은 줄에 작성된 주석을 말합니다. 인라인 주석은 명령문과 최소 두 칸의 공백으로 구분되어야 합니다.

Proper commenting에 대한 자세한 내용은 <a href="https://www.python.org/dev/peps/pep-0008/#comments">PEP 8 – Style Guide for Python Code</a>에서 확인할 수 있습니다.  

현재는 필요하지 않지만 삭제하고 싶지 않은 경우 코드 한 줄이나 블록을 주석 처리할 수도 있습니다.

주석 처리된 코드가 더 이상 강조 표시되지 않는 것을 확인하세요.
